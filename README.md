# Verdict AI

## AI-Powered Virtual Courtroom Platform

Verdict AI is an intelligent virtual courtroom platform designed to simulate legal proceedings using artificial intelligence. The platform enables users to participate in AI-assisted legal debates, case analysis, evidence presentation, and courtroom simulations through an interactive and modern web interface.

The project focuses on making legal education, case evaluation, and courtroom simulations more accessible through technology-driven workflows and AI-based reasoning systems.

## Installation

Clone the repository:

```bash
git clone https://github.com/Secret-moon128/Verdict-AI.git
```

Move into the frontend directory:

```bash
cd Verdict-AI/ai-courtroom-frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_API_URL=your_backend_url
VITE_AI_API_KEY=your_api_key
VITE_AUTH_KEY=your_authentication_key
```
